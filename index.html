Droid-X-HDMI-to-Monitor-Extension by JackHughes

This is an Android project to attempt to use the Droid X HDMI port to either extend the display onto and external monitor/tv OR mirror it. I intend to use this as a creative way to implement emulators and or design a new gaming system app (using the phone as both controller and system). Please join in!
